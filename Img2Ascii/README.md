Ini adalah aplikasi **Img2Ascii**, sebuah tool modern untuk mengubah gambar menjadi seni teks ASCII yang keren abis! Dibuat dengan cinta dan semangat 45 menggunakan **C#** dan **Avalonia UI**, jadi bisa jalan mulus di Windows, Linux, dan macOS.

## Fitur Keren
- **Image Upload**: Load gambar apa aja (PNG, JPG, BMP).
- **Resolution Control**: Atur detail ASCII art kamu dengan slider, makin gede makin detail!
- **Character Mapping**: Pilih gaya ASCII kamu (Standard, Complex, atau Minimal).
- **Invert Mode**: Cocok buat background terang atau gelap.
- **Export to Text**: Simpan hasil karya kamu ke file `.txt` buat dipamerin.
- **Modern UI**: Tampilan gelap (Dark Mode) dengan sentuhan Acrylic yang manis.

## Cara Pakai
1. Jalankan aplikasi.
2. Klik tombol **ðŸ“‚ Load Image** dan pilih gambar favoritmu.
3. Mainkan slider **Resolution** dan pilih **Character Set** sampai hasilnya pas di hati.
4. Kalau udah oke, klik **ðŸ’¾ Export to Text File** buat simpen.
5. Jangan lupa senyum!
