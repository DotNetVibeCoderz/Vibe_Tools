# PCInfo - Ultimate System Information Tool

![PCInfo](https://img.shields.io/badge/Status-Active-success) ![License](https://img.shields.io/badge/License-MIT-blue) ![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Linux%20%7C%20macOS-lightgrey)

**[English Description]**

PCInfo is a modern, cross-platform desktop application built with **C#** and **Avalonia UI**. It provides detailed specifications about your computer's hardware, operating system, and network configurations in a sleek, dark-themed interface.

## Features

*   **Dashboard**: Real-time overview of CPU, RAM usage, and GPU specifications.
*   **Storage Info**: Detailed list of connected drives, partitions, and available space.
*   **Advanced Network Stats**: 
    *   Internet Connectivity Status (Online/Offline).
    *   Domain and Hostname information.
    *   Detailed list of Network Adapters including IP Address (v4), Subnet Mask, Gateway, DNS Servers, and MAC Address.
*   **Cross-Platform**: Runs smoothly on Windows, Linux, and macOS.
*   **Modern UI**: Uses Avalonia's fluent theme with acrylic blur effects.

## How to Run

1.  Ensure you have **.NET 8.0 SDK** installed.
2.  Clone this repository.
3.  Open terminal in the project folder.
4.  Run the command:
    ```bash
    dotnet run
    ```
