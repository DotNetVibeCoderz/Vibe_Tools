# ContainerWatcher

**ContainerWatcher** adalah aplikasi manajemen Docker sederhana berbasis desktop yang dibangun menggunakan **C#** dan **Avalonia UI**. Aplikasi ini dirancang sebagai alternatif ringan untuk Docker Desktop dengan antarmuka yang modern dan mudah digunakan.

## Fitur Utama
- **Monitoring Container**: Melihat daftar container aktif dan non-aktif secara real-time.
- **Start/Stop Container**: Mengontrol status container dengan mudah.
- **Logs Viewer**: Melihat log output dari container langsung di aplikasi.
- **Cross-Platform**: Dapat berjalan di Windows, macOS, dan Linux (berkat Avalonia UI).

## Prasyarat
- **Docker Engine** atau **Docker Desktop** harus sudah terinstal dan berjalan di komputer Anda.
- **.NET 8 SDK** untuk menjalankan source code.

## Cara Menjalankan
1. Clone atau download repository ini.
2. Buka terminal di folder project.
3. Jalankan perintah:
   ```bash
   dotnet run
   ```

# ContainerWatcher (English)

**ContainerWatcher** is a simple Docker management desktop application built with **C#** and **Avalonia UI**. It serves as a lightweight alternative to Docker Desktop with a modern and user-friendly interface.

## Core Features
- **Container Monitoring**: View active and inactive containers in real-time.
- **Start/Stop Containers**: Easily control container status.
- **Logs Viewer**: View container logs directly within the app.
- **Cross-Platform**: Runs on Windows, macOS, and Linux (powered by Avalonia UI).

## Prerequisites
- **Docker Engine** or **Docker Desktop** must be installed and running.
- **.NET 8 SDK** to build and run the source code.

## How to Run
1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run the following command:
   ```bash
   dotnet run
   ```
