PLEASE READ:

To use this Midi Player, you need a SoundFont file (.sf2).
Because the automatic download failed, please:

1. Download "TimGM6mb.sf2" (or any other GM .sf2 file).
   Link: https://github.com/trevor0402/MeltySynth/blob/master/MeltySynth/Test/TimGM6mb.sf2
   (Click "Download" or "Raw")

2. Rename it to "TimGM6mb.sf2".

3. Place it in the "Assets" folder of this project.

The application will look for "Assets/TimGM6mb.sf2" on startup.
