# TerminalNet

TerminalNet is a modern, cross-platform terminal emulator built with C# and Avalonia UI. It provides a familiar command-line interface with modern UI enhancements.

## Features

*   **Cross-Platform**: Runs on Windows, Linux, and macOS.
*   **Modern UI**: Dark theme, syntax highlighting (simulated), and smooth tab management.
*   **Multi-Tab Support**: Open multiple terminal sessions in a single window.
*   **Command History**: Navigate through previous commands.
*   **Built-in Commands**: Includes standard commands like `ls`, `cd`, `cat`, `echo`, `mkdir`, `rm`, `curl`, `ping`, and more.
*   **Network Tools**: Integrated `ping`, `curl`, and `ifconfig`.

## Getting Started

### Prerequisites

*   .NET 8.0 SDK or later.

### Installation

1.  Clone the repository.
2.  Navigate to the project directory.
3.  Run the application:

```bash
dotnet run
```

## Supported Commands

| Command | Description |
| :--- | :--- |
| `help` | Show available commands |
| `ls` / `dir` | List files and directories |
| `cd <path>` | Change directory |
| `pwd` | Print working directory |
| `cat <file>` | Read file content |
| `echo <text>` | Display text |
| `clear` | Clear screen |
| `exit` | Close application |
| `history` | Show command history |
| `touch <file>` | Create empty file |
| `rm <file>` | Remove file |
| `mkdir <name>` | Create directory |
| `rmdir <name>` | Remove directory |
| `cp <src> <dst>` | Copy file |
| `mv <src> <dst>` | Move file |
| `whoami` | Show current user |
| `date` | Show date/time |
| `ping <host>` | Ping a host |
| `curl <url>` | Fetch URL content |
| `ifconfig` | Show network info |
| `version` | Show version |
