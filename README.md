# Ringkasan Proyek

## 1. Compressor - File Explorer

Aplikasi File Explorer sederhana yang dibangun menggunakan **Avalonia UI** dan **.NET**. Aplikasi ini memiliki fitur navigasi folder, manajemen file, serta kompresi dan ekstraksi arsip.

### Fitur Utama

*   **Navigasi Folder:** Pohon direktori (Tree View) di sebelah kiri untuk navigasi antar drive dan folder.
*   **Daftar File:** Menampilkan isi folder, termasuk nama, tanggal modifikasi, tipe, dan ukuran.
*   **Operasi File Dasar:**
    *   Membuat Folder Baru.
    *   Membuat File Baru (Txt, Doc, Xls).
    *   Menghapus File/Folder.
*   **Kompresi (Compress):** Mengkompres file atau folder ke dalam format ZIP.
*   **Ekstraksi (Extract):** Mengekstrak file arsip (ZIP, RAR, 7z, TAR, GZip) ke folder lokal.

## 2. Drawing App - MS Paint Clone v2.0

Aplikasi menggambar lengkap seperti MS Paint yang dibuat dengan **Avalonia UI** - Cross-platform (Windows, Linux, Mac)!

### Fitur Utama

*   **Drawing Tools (7 Tools)**: Pencil, Brush, Line, Rectangle, Circle, Arrow, Eraser.
*   **Color System**: 12 Warna Palette, Current Color Display, Visual Feedback.
*   **File Operations**: Save As, New File, Status Feedback.

## 3. Image Explorer - Code Bender Edition

Aplikasi desktop cross-platform yang ringan dan cepat untuk menjelajahi, melihat, dan mengedit gambar.

### Fitur Utama

*   **File Explorer Terintegrasi**: Navigasi folder dengan tampilan Tree View.
*   **Viewer Canggih**: Zooming, Auto Fit, Panning.
*   **Editor Gambar**: Resize, Rotate, Filter Efek.

## 4. Network Scanner CLI

Alat pemindai jaringan sederhana namun kuat yang dibangun menggunakan C# dan .NET.

### Fitur Utama

*   **Pemindaian Subnet**: Memindai rentang IP, mendeteksi status Online/Offline.
*   **Pemindaian Port Host**: Memeriksa ketersediaan host tertentu.
*   **Wake-on-LAN (WOL)**: Mengirimkan "Magic Packet" untuk menyalakan komputer.

## 5. PCInfo - Ultimate System Information Tool

Aplikasi desktop modern dan cross-platform yang menampilkan informasi rinci mengenai spesifikasi hardware komputer.

### Fitur Utama

*   **Dashboard**: Ringkasan real-time mengenai CPU, RAM, dan spesifikasi GPU.
*   **Informasi Penyimpanan**: Daftar lengkap hard drive, partisi, dan kapasitas penyimpanan.

## 6. WeatherApp

Aplikasi cuaca desktop lintas platform yang mengambil data cuaca real-time dari **Open-Meteo**.

### Fitur Utama

*   **Cuaca real-time** untuk kota-kota preset.
*   **Dynamic background** dan icons berdasarkan cuaca dan siang/malam.
