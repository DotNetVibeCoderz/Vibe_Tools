# StrudelNet

StrudelNet is a Desktop Webview Music Player and Editor designed for live coding using the Strudel (TidalCycles in JavaScript) engine. It provides a hybrid WPF + Blazor interface for a seamless desktop experience.

## Features

- **Live Coding Interface (REPL)**: Write and run code interactively to generate music.
- **Sample Library**: Includes 20 default music patterns to get you started.
- **File Management**: Open and Save your `.strudel` scripts directly.
- **Desktop Integration**: Runs as a native Windows application.

## Requirements

- .NET 8.0 Runtime (or SDK to build)
- Windows 10/11 (WebView2 runtime is usually pre-installed)

## How to Use

1. **Launch**: Open `StrudelNet.exe`.
2. **Play**: Press `Ctrl+Enter` or click the "Play" button to run the code.
3. **Stop**: Press `Ctrl+.` or click "Stop" to silence the audio.
4. **Load Samples**: Click on any sample in the sidebar to load it into the editor.
5. **Save/Open**: Use the buttons at the bottom of the sidebar.

# StrudelNet (Bahasa Indonesia)

StrudelNet adalah Pemutar Musik dan Editor Webview Desktop yang dirancang untuk live coding menggunakan mesin Strudel (TidalCycles dalam JavaScript). Aplikasi ini menyediakan antarmuka hybrid WPF + Blazor untuk pengalaman desktop yang mulus.

## Fitur

- **Antarmuka Live Coding (REPL)**: Tulis dan jalankan kode secara interaktif untuk menghasilkan musik.
- **Pustaka Sampel**: Termasuk 20 pola musik default untuk memulai.
- **Manajemen File**: Buka dan Simpan skrip `.strudel` Anda secara langsung.
- **Integrasi Desktop**: Berjalan sebagai aplikasi Windows native.

## Persyaratan

- .NET 8.0 Runtime (atau SDK untuk membangun)
- Windows 10/11 (WebView2 runtime biasanya sudah terinstal)

## Cara Menggunakan

1. **Buka**: Jalankan `StrudelNet.exe`.
2. **Mainkan**: Tekan `Ctrl+Enter` atau klik tombol "Play" untuk menjalankan kode.
3. **Berhenti**: Tekan `Ctrl+.` atau klik "Stop" untuk mematikan audio.
4. **Muat Sampel**: Klik pada sampel mana pun di sidebar untuk memuatnya ke dalam editor.
5. **Simpan/Buka**: Gunakan tombol di bagian bawah sidebar.
